# repository_1
# 📦 Бот-каталог

Простой Telegram-бот на Python для листалки страниц с Inline-кнопками.

---

## 🚀 Возможности

- 📄 Листалка с тремя страницами  
- ⬅️➡️ Inline-кнопки "Вперёд" и "Назад"  
- 🧩 Команды: `/start`, `/pages`  
- 🛠 Простая структура для начинающих

---

## 🛠 Технологии

- Python 3.13  
- Aiogram 3.29.0  
- InlineKeyboard

---
